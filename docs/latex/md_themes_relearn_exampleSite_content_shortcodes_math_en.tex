The {\ttfamily math} shortcode generates beautiful formatted math and chemical formulae using the \href{https://mathjax.org/}{\texttt{ Math\+Jax}} library.

\{\{$<$ math align=\char`\"{}center\char`\"{} $>$\}\} \$\$\textbackslash{}left( \textbackslash{}sum\+\_\+\{k=1\}$^\wedge$n a\+\_\+k b\+\_\+k \textbackslash{}right)$^\wedge$2 \textbackslash{}leq \textbackslash{}left( \textbackslash{}sum\+\_\+\{k=1\}$^\wedge$n a\+\_\+k$^\wedge$2 \textbackslash{}right) \textbackslash{}left( \textbackslash{}sum\+\_\+\{k=1\}$^\wedge$n b\+\_\+k$^\wedge$2 \textbackslash{}right)\$\$ \{\{$<$ /math $>$\}\}\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1086}{}\doxysubsection{Usage}\label{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1086}
While the examples are using shortcodes with named parameter it is recommended to use codefences instead. This is because more and more other software supports Math codefences (eg. Git\+Hub) and so your markdown becomes more portable.

You are free to also call this shortcode from your own partials.

Math is also usable without enclosing it in a shortcode or codefence but \href{\#passthrough-configuration}{\texttt{ requires configuration}} by you. In this case no parameter from the below table are available.

\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1088}{}\doxysubsection{Configuration}\label{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1088}
Math\+Jax is configured with default settings but you can customize Math\+Jax\textquotesingle{}s default settings for all of your files through a JSON object in your {\ttfamily hugo.\+toml} or override these settings per page through your pages frontmatter.

The JSON object of your {\ttfamily hugo.\+toml} / frontmatter is forwarded into Math\+Jax\textquotesingle{}s configuration object.

See \href{https://docs.mathjax.org/en/latest/options/index.html}{\texttt{ Math\+Jax documentation}} for all allowed settings.\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1089}{}\doxysubsubsection{Global Configuration File}\label{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1089}
This example reflects the default configuration also used if you don\textquotesingle{}t define {\ttfamily math\+Jax\+Initialize}

\{\{$<$ multiconfig file=hugo $>$\}\} \mbox{[}params\mbox{]} math\+Jax\+Initialize = \char`\"{}\{ \textbackslash{}\char`\"{}tex"{}\+: \{ "{}inline\+Math"{}\+: \mbox{[}\mbox{[}"{}\textbackslash{}("{}, "{}\textbackslash{})"{}\mbox{]}, \mbox{[}"{}\$"{}, "{}\$"{}\mbox{]}\mbox{]}, display\+Math\+: \mbox{[}\mbox{[}"{}\textbackslash{}\mbox{[}"{}, "{}\textbackslash{}\mbox{]}"{}\mbox{]}, \mbox{[}"{}\$\$"{}, "{}\$\$"{}\mbox{]}\mbox{]} \}, "{}options"{}\+: \{ "{}enable\+Menu"{}\+: false \}"{} \{\{$<$ /multiconfig $>$\}\}\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1090}{}\doxysubsubsection{Page\textquotesingle{}s Frontmatter}\label{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1090}
Usually you don\textquotesingle{}t need to redefine the global initialization settings for a single page. But if you do, you have repeat all the values from your global configuration you want to keep for a single page as well.

Eg. If you have redefined the delimiters to something exotic like {\ttfamily @} symbols in your global config, but want to additionally align your math to the left for a specific page, you have to put this to your frontmatter\+:

\{\{$<$ multiconfig fm=true $>$\}\} math\+Jax\+Initialize = \char`\"{}\{ \textbackslash{}\char`\"{}chtml"{}\+: \{ "{}display\+Align"{}\+: "{}left"{} \}, \{ "{}tex"{}\+: \{ "{}inline\+Math"{}\+: \mbox{[}\mbox{[}"{}\textbackslash{}("{}, "{}\textbackslash{})"{}\mbox{]}, \mbox{[}"{}\textbackslash{}\char`\"{}, \textbackslash{}\char`\"{}\textbackslash{}\char`\"{}\mbox{]}\mbox{]}, display\+Math\+: \mbox{[}\mbox{[}\textbackslash{}\char`\"{}\textbackslash{}\mbox{[}"{}, "{}\textbackslash{}\mbox{]}"{}\mbox{]}, \mbox{[}"{}@"{}, "{}@"{}\mbox{]}\mbox{]} \}, "{}options"{}\+: \{ "{}enable\+Menu"{}\+: false \}"{} \{\{$<$ /multiconfig $>$\}\}\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1091}{}\doxysubsubsection{Passthrough Configuration}\label{md_themes_relearn_exampleSite_content_shortcodes_math_en_autotoc_md1091}
You can use your math without enclosing it in a shortcode or codefence by using a \href{https://gohugo.io/content-management/mathematics/\#step-1}{\texttt{ passthrough configuration}} in your {\ttfamily hugo.\+toml}\+:

\{\{$<$ multiconfig file=hugo $>$\}\} \mbox{[}markup\mbox{]} \mbox{[}markup.\+goldmark\mbox{]} \mbox{[}markup.\+goldmark.\+extensions\mbox{]} \mbox{[}markup.\+goldmark.\+extensions.\+passthrough\mbox{]} enable = true \mbox{[}markup.\+goldmark.\+extensions.\+passthrough.\+delimiters\mbox{]} inline = \mbox{[}\mbox{[}\textquotesingle{}(\textquotesingle{}, \textquotesingle{})\textquotesingle{}\mbox{]}, \mbox{[}\textquotesingle{}\$\textquotesingle{}, \textquotesingle{}\$\textquotesingle{}\mbox{]}\mbox{]} block = \mbox{[}\mbox{[}\textquotesingle{}\mbox{[}\textquotesingle{}, \textquotesingle{}\mbox{]}\textquotesingle{}\mbox{]}, \mbox{[}\textquotesingle{}\$\$\textquotesingle{}, \textquotesingle{}\$\$\textquotesingle{}\mbox{]}\mbox{]} \{\{$<$ /multiconfig $>$\}\}

In this case you have to tell the theme that your page contains math by setting this in your page\textquotesingle{}s frontmatter\+:

\{\{$<$ multiconfig fm=true $>$\}\} disable\+Math\+Jax = false \{\{$<$ /multiconfig $>$\}\}
