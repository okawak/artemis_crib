\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_icon_en_autotoc_md1066}{}\doxysubsubsection{Finding an icon}\label{md_themes_relearn_exampleSite_content_shortcodes_icon_en_autotoc_md1066}
Browse through the available icons in the \href{https://fontawesome.com/v6/search?m=free}{\texttt{ Font Awesome Gallery}}. Notice that the {\bfseries{free}} filter is enabled, as only the free icons are available by default.

Once on the Font Awesome page for a specific icon, for example the page for the \href{https://fontawesome.com/v6/icons/heart?s=solid}{\texttt{ heart}}, copy the icon name and paste into the Markdown content.\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_icon_en_autotoc_md1067}{}\doxysubsubsection{Customising Icons}\label{md_themes_relearn_exampleSite_content_shortcodes_icon_en_autotoc_md1067}
Font Awesome provides many ways to modify the icon


\begin{DoxyItemize}
\item Change color (by default the icon will inherit the parent color)
\item Increase or decrease size
\item Rotate
\item Combine with other icons
\end{DoxyItemize}
