Default constructor. 

From telescope object and beam (tracking) object, it calculate the reaction position (event reconstruction) using Thick Gas Target method. (Thick Gas Target Inverse Kinematics \+: TGTIK)

Need variables in the steering file\+:
\begin{DoxyItemize}
\item Input\+Collection\+: telescope object, inherit from \mbox{\hyperlink{classart_1_1TTelescopeData}{art\+::\+TTelescope\+Data}}
\item Input\+Track\+Collection\+: tracking object, inherit from art\+::\+TTrack
\item Output\+Collection\+: name of output branch, inherit from \mbox{\hyperlink{classart_1_1TReactionInfo}{art\+::\+TReaction\+Info}}
\item Initial\+Beam\+Energy\+: beam energy in LAB system in MeV unit
\item Target\+Name\+: target gas name for TSrim input
\item Target\+Pressure\+: gas pressure in Torr unit
\item Target\+Temperature\+: gas emperature
\item Particle\+ZArray\+: \mbox{[}Z1, Z2, Z3, Z4\mbox{]}
\item Particle\+AArray\+: \mbox{[}A1, A2, A3, A4\mbox{]}
\begin{DoxyItemize}
\item assuming two body reaction (NEED size 4 aray)
\item \mbox{[}beam, target, fragment, residual\mbox{]}
\item example\+: A(a,b)B reaction,
\item the order is \mbox{[}a, A, b, B\mbox{]}
\item this is \char`\"{}inverse kinematics\char`\"{}, so we assume the detected particle is \char`\"{}\+B\char`\"{}
\end{DoxyItemize}
\end{DoxyItemize}

Option\+:
\begin{DoxyItemize}
\item Excited\+Energy\+: if you want to treat excited state transition, input the excited energy (MeV) of Z3 particles
\item Use\+Custom\+Function\+: if you need extra treatment, you can add the process.
\begin{DoxyItemize}
\item if it is \char`\"{}true\char`\"{}, the custom treatment is on.
\item this is for 26Si(a, p)29P analysis to treat the excited state effect. 
\end{DoxyItemize}
\end{DoxyItemize}
