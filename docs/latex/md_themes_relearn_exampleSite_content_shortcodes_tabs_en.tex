The {\ttfamily tabs} shortcode displays arbitrary content in an unlimited number of tabs.

This comes in handy eg. for providing code snippets for multiple languages.

If you just want a single tab you can instead call the \href{shortcodes/tab}{\texttt{ {\ttfamily tab} shortcode}} standalone.

~\newline

In case you want to nest tab views, the parent tab that contains nested tab views needs to be declared with {\ttfamily \{\{$<$/$\ast$ tab $\ast$/$>$\}\}} instead of {\ttfamily \{\{\%/$\ast$ tab $\ast$/\%\}\}}. Note, that in this case it is not possible to put markdown in the parent tab.

You can also set style and color parameter for all tabs and overwrite them on tab level. See the \href{shortcodes/tab\#parameter}{\texttt{ {\ttfamily tab} shortcode}} for possible values.
