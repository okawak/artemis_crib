The {\ttfamily mermaid} shortcode generates diagrams and flowcharts from text, in a similar manner as Markdown using the \href{https://mermaidjs.github.io/}{\texttt{ Mermaid}} library.

While the examples are using shortcodes with named parameter it is recommended to use codefences instead. This is because more and more other software supports Mermaid codefences (eg. Git\+Hub) and so your markdown becomes more portable.

You are free to also call this shortcode from your own partials.

Mermaid is configured with default settings. You can customize Mermaid\textquotesingle{}s default settings for all of your files through a JSON object in your {\ttfamily hugo.\+toml}, override these settings per page through your pages frontmatter or override these setting per diagramm through \href{https://mermaid-js.github.io/mermaid/\#/directives?id=directives}{\texttt{ diagram directives}}.

The JSON object of your {\ttfamily hugo.\+toml} / frontmatter is forwarded into Mermaid\textquotesingle{}s {\ttfamily mermaid.\+initialize()} function.

See \href{https://mermaid-js.github.io/mermaid/\#/Setup?id=mermaidapi-configuration-defaults}{\texttt{ Mermaid documentation}} for all allowed settings.

The {\ttfamily theme} setting can also be set by your used color variant.
