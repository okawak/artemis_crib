{\footnotesize\ttfamily TReconst\+Processor\+::\+TReconst\+Processor (\begin{DoxyParamCaption}{ }\end{DoxyParamCaption})}



Default constructor. 

From telescope object and beam (tracking) object, it calculate the Ecm assuming reaction position is z = 0 for solid target.

Need variables in the steering file\+:
\begin{DoxyItemize}
\item Input\+Collection\+: telescope object, inherit from art\+::\+TTelescope\+Data
\item Input\+Track\+Collection\+: tracking object, inherit from art\+::\+TTrack
\item Output\+Collection\+: name of output branch, inherit from art\+::\+TReaction\+Info
\item Particle\+ZArray\+: \mbox{[}Z1, Z2, Z3, Z4\mbox{]}
\item Particle\+AArray\+: \mbox{[}A1, A2, A3, A4\mbox{]}
\begin{DoxyItemize}
\item assuming two body reaction (NEED size 4 aray)
\item \mbox{[}beam, target, fragment, residual\mbox{]}
\item example\+: A(a,b)B reaction,
\item the order is \mbox{[}a, A, b, B\mbox{]}
\item this is \char`\"{}inverse kinematics\char`\"{}, so we assume the detected particle is \char`\"{}\+B\char`\"{}
\end{DoxyItemize}
\end{DoxyItemize}

Option\+:
\begin{DoxyItemize}
\item Excited\+Energy\+: if you want to treat excited state transition, input the excited energy (MeV) of Z3 particles
\item Use\+Center\+Position\+: use center position at the detector.
\begin{DoxyItemize}
\item if DSSSD is not working, this flag is used. 
\end{DoxyItemize}
\end{DoxyItemize}
