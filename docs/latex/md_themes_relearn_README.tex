\hypertarget{md_themes_relearn_README_autotoc_md1189}{}\doxysection{Motivation}\label{md_themes_relearn_README_autotoc_md1189}
The Relearn theme is a fork of the great \href{https://github.com/matcornic/hugo-theme-learn}{\texttt{ Learn theme}} with the aim of fixing long outstanding bugs and adapting to latest Hugo features. As far as possible this theme tries to be a drop-\/in replacement for the Learn theme.\hypertarget{md_themes_relearn_README_autotoc_md1190}{}\doxysection{Features}\label{md_themes_relearn_README_autotoc_md1190}

\begin{DoxyItemize}
\item {\bfseries{Wide set of usage scenarios}}
\begin{DoxyItemize}
\item Responsive design for mobile usage
\item Looks nice on paper (if it has to)
\item Usable offline, no external dependencies
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/customization\#file-system}{\texttt{ Usable from your local file system via {\ttfamily file\+://} protocol}}
\item Support for the \href{https://github.com/estruyf/vscode-front-matter}{\texttt{ VSCode Front Matter extension}} for on-\/premise CMS capabilities
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/customization\#social-media-meta-tags}{\texttt{ Support for Open Graph, Schema and Twitter Cards}}
\end{DoxyItemize}
\item {\bfseries{Configurable theming and visuals}}
\begin{DoxyItemize}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/branding\#change-the-logo}{\texttt{ Configurable brand images}}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/branding\#adjust-to-os-settings}{\texttt{ Automatic switch for light/dark variant depending on your OS settings}}
\item Predefined light, dark and color variants
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/branding\#multiple-variants}{\texttt{ User selectable variants}}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/generator}{\texttt{ Stylesheet generator}}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/shortcodes/highlight}{\texttt{ Configurable syntax highlighting}}
\end{DoxyItemize}

\item \href{https://mcshelby.github.io/hugo-theme-relearn/basics/customization\#activate-dedicated-search-page}{\texttt{ Dedicated search page}}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/cont/taxonomy}{\texttt{ Taxonomy support}}

\item \href{https://mcshelby.github.io/hugo-theme-relearn/cont/pages}{\texttt{ Unlimited nested menu items}}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/cont/menushortcuts}{\texttt{ Configurable shortcut links}}
\item Hidden pages

\item {\bfseries{Multi language support}}
\begin{DoxyItemize}
\item \href{https://mcshelby.github.io/hugo-theme-relearn/cont/i18n}{\texttt{ Full support for languages written right to left}}
