The {\ttfamily attachments} shortcode displays a list of files attached to a page with adjustable color, title and icon.\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md977}{}\doxysubsection{Migration}\label{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md977}
While this shortcode will still be available for some time, it does not receive support anymore. Start to migrate early, as it will be removed with the next major update of the theme.

The \href{shortcodes/resources}{\texttt{ {\ttfamily resources} shortcode}} leverages Hugo\textquotesingle{}s resource feature for page bundles. It has all the same parameter as the {\ttfamily attachments} shortcode but applies the {\ttfamily pattern} directly on a resources {\ttfamily Name} attribute.

To migrate your pages apply the following steps\+:


\begin{DoxyEnumerate}
\item If a page is not already a \href{https://gohugo.io/content-management/page-bundles/}{\texttt{ page bundle}} convert it
\item Move your files to a valid destination inside of your page bundle (depending if you have a branch or a leaf bundle)
\item Change the calls from the {\ttfamily attachments} shortcode to the \href{shortcodes/resources}{\texttt{ {\ttfamily resources} shortcode}} and adjust the {\ttfamily pattern} parameter to the new directory layout and the resources \href{https://gohugo.io/methods/resource/name/}{\texttt{ {\ttfamily Name} attribute}}.
\end{DoxyEnumerate}

Multilanguage features are not supported directly by the shortcode anymore but rely on Hugo\textquotesingle{}s handling for resource translations.\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md978}{}\doxysubsection{Usage}\label{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md978}
While the examples are using shortcodes with named parameter you are free to also call this shortcode from your own partials.

\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md980}{}\doxysubsection{Setup}\label{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md980}
\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md981}{}\doxysubsubsection{Single language}\label{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md981}
The shortcode lists files found in a specific folder. The name of the folder depends on your page type (either branch bundle, leaf bundle or page).


\begin{DoxyEnumerate}
\item If your page is a leaf bundle, attachments must be placed in a nested {\ttfamily index.\+files} folder, accordingly.

$>$ $\ast$ content $>$ $\ast$ \+\_\+index.\+md $>$ $\ast$ page $>$ $\ast$ index.\+md $>$ $\ast$ {\bfseries{index.\+files}} $>$ $\ast$ attachment.\+pdf
\item If your page is a branch bundle, attachments must be placed in a nested {\ttfamily \+\_\+index.\+files} folder, accordingly.

\{\{\% badge style=\char`\"{}warning\char`\"{} title=\char`\"{} \char`\"{} \%\}\}Warning\{\{\% /badge \%\}\} This is only available for Hugo $<$ {\ttfamily 0.\+112.\+0}

$>$ $\ast$ content $>$ $\ast$ \+\_\+index.\+md $>$ $\ast$ page $>$ $\ast$ \+\_\+index.\+md $>$ $\ast$ $\ast$$\ast$\+\_\+index.files$\ast$$\ast$ $>$ $\ast$ attachment.\+pdf
\item For simple pages, attachments must be placed in a folder named like your page and ending with {\ttfamily .files}.

\{\{\% badge style=\char`\"{}warning\char`\"{} title=\char`\"{} \char`\"{} \%\}\}Warning\{\{\% /badge \%\}\} This is only available for Hugo $<$ {\ttfamily 0.\+112.\+0}

$>$ $\ast$ content $>$ $\ast$ \+\_\+index.\+md $>$ $\ast$ {\bfseries{page.\+files}} $>$ $\ast$ attachment.\+pdf $>$ $\ast$ page.\+md
\end{DoxyEnumerate}\hypertarget{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md982}{}\doxysubsubsection{Multilingual}\label{md_themes_relearn_exampleSite_content_shortcodes_attachments_index_en_autotoc_md982}
Be aware that if you use a multilingual website, you will need to have as many folders as languages and the language code must be part of the folder name.
