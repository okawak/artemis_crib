If you forgot to \char`\"{}git pull\char`\"{} and you have already modified the files, you will get conflicts. If you think it is okay to forget your modified file, please command like 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ git stash}

\end{DoxyCode}
 and 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ git pull}

\end{DoxyCode}
 And you want to recover your modified file, 
\begin{DoxyCode}{0}
\DoxyCodeLine{\$ git stash pop}

\end{DoxyCode}
 Then you encounter the conficts again, and please select the code you want to keep, and git push again. For the detail, please check \href{../example/preparation/git}{\texttt{ here}}.
