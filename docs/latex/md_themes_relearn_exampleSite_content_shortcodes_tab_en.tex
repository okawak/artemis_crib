You can use a {\ttfamily tab} shortcode to display a single tab.

This is especially useful if you want to flag your code example with an explicit language.

If you want multiple tabs grouped together you can wrap your tabs into the \href{shortcodes/tabs}{\texttt{ {\ttfamily tabs} shortcode}}.

\{\{$<$ tabs $>$\}\} \{\{\% tab title=\char`\"{}just colored style\char`\"{} style=\char`\"{}blue\char`\"{} \%\}\}

The {\ttfamily style} parameter is set to a color style.

This will set the background to a lighter version of the chosen style color as configured in your theme variant.

\{\{\% /tab \%\}\} \{\{\% tab title=\char`\"{}just color\char`\"{} color=\char`\"{}blue\char`\"{} \%\}\}

Only the {\ttfamily color} parameter is set.

This will set the background to a lighter version of the chosen CSS color value.

\{\{\% /tab \%\}\} \{\{\% tab title=\char`\"{}default style and color\char`\"{} style=\char`\"{}default\char`\"{} color=\char`\"{}blue\char`\"{} \%\}\}

The {\ttfamily style} parameter affects how the {\ttfamily color} parameter is applied.

The {\ttfamily default} style will set the background to your {\ttfamily -\/-\/MAIN-\/\+BG-\/color} as configured for your theme variant resembling the default style but with different color.

\{\{\% /tab \%\}\} \{\{\% tab title=\char`\"{}just severity style\char`\"{} style=\char`\"{}info\char`\"{} \%\}\}

The {\ttfamily style} parameter is set to a severity style.

This will set the background to a lighter version of the chosen style color as configured in your theme variant and also affects the chosen icon.
